# Monocular Depth Estimation for Mini Robot Navigation

This project implements monocular depth estimation using the MiDaS model for obstacle detection and navigation of a mini robot.

## Features

- Real-time depth estimation
- Webcam and video inference
- Quantitative calibration
- Obstacle detection

## Setup

Clone repository

git clone https://github.com/USERNAME/monocular-depth-mini-robot.git

Install dependencies

pip install -r requirements.txt

Run demo

python inference/depth_webcam_demo.py
